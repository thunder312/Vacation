# Benutzerhandbuch: Mitarbeiter (Employee)

## Übersicht

Als Mitarbeiter können Sie das Urlaubsverwaltungssystem nutzen, um:
- Urlaubsanträge einzureichen
- Den Status Ihrer Anträge zu verfolgen
- Ihr Urlaubskonto einzusehen
- Den Urlaubskalender des Teams zu sehen

---

## Anmeldung

1. Öffnen Sie die Anwendung in Ihrem Browser
2. Geben Sie Ihren **Benutzernamen** ein
3. Geben Sie Ihr **Passwort** ein
4. Klicken Sie auf **Anmelden**

> **Tipp:** Beim ersten Login sollten Sie Ihr Passwort ändern.

---

## Hauptbereiche

Nach der Anmeldung sehen Sie folgende Tabs:

| Tab | Beschreibung |
|-----|--------------|
| **Mein Antrag** | Urlaubsanträge stellen und verwalten |
| **Organisation** | Organisationsstruktur einsehen |
| **Kalender** | Urlaubskalender des Teams |

---

## Urlaubsantrag stellen

### Schritt 1: Tab "Mein Antrag" öffnen

Dieser Tab ist standardmäßig nach dem Login ausgewählt.

### Schritt 2: Urlaubskonto prüfen

Oben sehen Sie Ihr aktuelles Urlaubskonto:

```
┌─────────────────────────────────────┐
│  Urlaubskonto                       │
│  ─────────────────────────────────  │
│  Gesamt:     32 Tage                │
│  Genommen:   10 Tage                │
│  Verbleibend: 22 Tage               │
│  (inkl. 2 Tage Übertrag aus 2025)   │
└─────────────────────────────────────┘
```

### Schritt 3: Neuen Antrag ausfüllen

1. **Von-Datum:** Wählen Sie das Startdatum Ihres Urlaubs
2. **Bis-Datum:** Wählen Sie das Enddatum (oder leer lassen für einzelnen Tag)
3. **Grund (optional):** Geben Sie einen Grund an

### Schritt 4: Antrag absenden

Klicken Sie auf **Antrag absenden**.

> **Hinweis:** Wenn Sie mehr Tage beantragen als Sie haben, erscheint eine Warnung. Sie können trotzdem fortfahren.

---

## Antragsstatus verstehen

Ihre Anträge können folgende Status haben:

| Status | Bedeutung | Farbe |
|--------|-----------|-------|
| **Ausstehend** | Warten auf Genehmigung durch Teamleiter | Gelb |
| **Teamleiter genehmigt** | Teamleiter hat genehmigt, warten auf Manager | Blau |
| **Genehmigt** | Vollständig genehmigt | Grün |
| **Abgelehnt** | Der Antrag wurde abgelehnt | Rot |
| **Storniert** | Der Urlaub wurde nachträglich storniert | Grau |
| **Zurückgezogen** | Sie haben den Antrag zurückgezogen | Grau |

---

## Antrag zurückziehen

Sie können einen **noch nicht genehmigten** Antrag zurückziehen:

1. Finden Sie den Antrag in der Liste "Meine Urlaubsanträge"
2. Klicken Sie auf **Zurückziehen**
3. Bestätigen Sie die Aktion

> **Achtung:** Genehmigte Anträge können nur vom Manager storniert werden.

---

## Urlaubskalender nutzen

Im Tab **Kalender** sehen Sie:

- **Grün:** Genehmigte Urlaube von Kollegen
- **Grau:** Wochenenden
- **Rot:** Feiertage
- **Gelb:** Halbe Arbeitstage (z.B. Heiligabend)

### Monatsnavigation

- Klicken Sie auf **◀** oder **▶** um zwischen Monaten zu wechseln
- Klicken Sie auf **Heute**, um zum aktuellen Monat zurückzukehren

---

## PDF-Export

Sie können Ihre genehmigten Urlaube als PDF exportieren:

1. Gehen Sie zum Tab **Mein Antrag**
2. Scrollen Sie zum Bereich "Genehmigte Urlaube exportieren"
3. Klicken Sie auf **Als PDF exportieren**

---

## Passwort ändern

1. Klicken Sie oben rechts auf Ihren Benutzernamen
2. Wählen Sie **Passwort ändern**
3. Geben Sie Ihr altes Passwort ein
4. Geben Sie das neue Passwort ein (mind. 8 Zeichen)
5. Bestätigen Sie das neue Passwort
6. Klicken Sie auf **Passwort ändern**

---

## Sprache ändern

Die Anwendung unterstützt mehrere Sprachen:

1. Klicken Sie auf das Sprachsymbol oben rechts
2. Wählen Sie: **DE** (Deutsch), **EN** (Englisch) oder **PT** (Portugiesisch)

---

## Design ändern (Hell/Dunkel)

1. Klicken Sie auf das Sonnen-/Mond-Symbol in der Tab-Leiste
2. Das Design wechselt zwischen Hell und Dunkel

---

## Häufige Fragen (FAQ)

### Wie viele Urlaubstage habe ich?

Ihr Urlaubskonto wird im Tab "Mein Antrag" oben angezeigt. Standard sind 30 Tage pro Jahr.

### Was passiert mit meinem Resturlaub?

Resturlaub wird am Jahresanfang automatisch ins neue Jahr übertragen. Ihr Manager führt den Jahreswechsel durch.

### Kann ich einen halben Tag Urlaub nehmen?

Ja, an bestimmten Tagen (z.B. Heiligabend) wird automatisch nur ein halber Tag abgezogen.

### Wer genehmigt meinen Urlaub?

1. Zuerst Ihr Teamleiter
2. Dann der Manager für die finale Genehmigung

### Kann ich meinen Antrag ändern?

Nein, Sie müssen den Antrag zurückziehen und einen neuen stellen.
