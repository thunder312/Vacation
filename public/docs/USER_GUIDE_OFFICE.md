# Benutzerhandbuch: Office / Sekretariat

## Übersicht

Als Office-Benutzer haben Sie **Leserechte** auf alle Bereiche des Systems. Sie können:
- Eigene Urlaubsanträge stellen
- Alle Urlaubsanträge einsehen (ohne Genehmigungsrecht)
- Berichte erstellen und exportieren
- Den Urlaubskalender einsehen
- Das Organigramm einsehen

---

## Ihre Tabs

| Tab | Beschreibung |
|-----|--------------|
| **Mein Antrag** | Eigene Urlaubsanträge verwalten |
| **Teamleiter-Ansicht** | Alle Teamleiter-Anträge einsehen (nur lesen) |
| **Manager-Ansicht** | Alle Manager-Anträge einsehen (nur lesen) |
| **Urlaubsregelung** | Halbtags-Regeln einsehen (nur lesen) |
| **Organisation** | Organigramm einsehen |
| **Berichte** | Jahresberichte erstellen |
| **Kalender** | Urlaubskalender aller Mitarbeiter |

---

## Tab "Mein Antrag"

### Eigenen Urlaub beantragen

Als Office-Mitarbeiter haben Sie keinen Teamleiter. Ihr Antrag geht direkt zum Manager:

1. Wählen Sie **Von-Datum** und **Bis-Datum**
2. Optional: Geben Sie einen **Grund** an
3. Klicken Sie auf **Antrag absenden**

### Ihr Urlaubskonto

Oben im Tab sehen Sie Ihr aktuelles Kontingent:
- **Gesamt:** Ihre Urlaubstage inkl. Übertrag
- **Genommen:** Bereits genehmigte/genommene Tage
- **Verbleibend:** Noch verfügbare Tage

---

```
│  Status: Warten auf Teamleiter-Genehmigung             │
└─────────────────────────────────────────────────────────┘
```

> **Hinweis:** Sie können diese Anträge nur **einsehen**, nicht genehmigen.

## Tab "Manager-Ansicht"

Hier sehen Sie:
1. **Anträge zur Endgenehmigung:** Warten auf Manager
2. **Genehmigte Urlaube:** Bereits bestätigte Urlaube

### PDF-Export

Auch ohne Genehmigungsrecht können Sie PDFs exportieren:
- **Ungenehmigte Urlaube exportieren:** Liste aller offenen Anträge

## Tab "Urlaubsregelung"

Hier sehen Sie die definierten Halbtags-Regeln:

| Datum | Beschreibung |
|-------|--------------|
| 24.12.2026 | Heiligabend |
| 31.12.2026 | Silvester |

> **Hinweis:** Sie können diese Regeln nur **einsehen**, nicht bearbeiten. Wenden Sie sich an den Manager für Änderungen.

---

## Tab "Berichte"

Sie haben **vollen Zugriff** auf die Berichtsfunktion:

### Jahresbericht erstellen

1. Wählen Sie das **Jahr** aus der Dropdown-Liste
2. Die Vorschau zeigt:
   - Anzahl aktiver Mitarbeiter
   - Gesamte Urlaubstage
   - Durchschnitt pro Mitarbeiter
3. Klicken Sie auf **Jahresbericht als PDF erstellen**

### Inhalt des Jahresberichts

- **Seite 1:** Gesamtübersicht mit Statistiken
- **Ab Seite 2:** Einzelblätter pro Mitarbeiter
  - Name und Abteilung
  - Urlaubsanspruch und Übertrag
  - Liste aller Urlaube
  - Unterschriftsfeld

---

## Tab "Organisation"

### Organigramm einsehen

Sie sehen die komplette Organisationsstruktur:
- Manager an der Spitze
- Teamleiter darunter
- Mitarbeiter in ihren Teams

### PDF-Export

Klicken Sie auf **Organigramm als PDF exportieren** für eine druckbare Version.

---

## Tab "Kalender"

### Funktionen

- **Monatsansicht:** Übersicht aller Urlaube
- **Navigation:** Vor/Zurück zwischen Monaten
- **Legende:**
  - Grün = Genehmigter Urlaub
  - Grau = Wochenende
  - Rot = Feiertag
  - Gelb = Halber Arbeitstag

### PDF-Export

- **Monat exportieren:** Aktuellen Monat als PDF
- **Jahr exportieren:** Ganzes Jahr als PDF

## Typische Aufgaben für Office

### 1. Urlaubsübersicht erstellen

Wenn Kollegen nach dem Urlaubsstand fragen:
1. Öffnen Sie den **Kalender**
2. Navigieren Sie zum gewünschten Zeitraum
3. Oder exportieren Sie einen **Monatsbericht**

### 2. Jahresabschluss unterstützen

Am Jahresende:
1. Erstellen Sie den **Jahresbericht** für die Dokumentation
2. Drucken Sie die Einzelblätter zur Unterschrift
3. Archivieren Sie die unterschriebenen Dokumente

### 3. Neue Mitarbeiter informieren

Bei Neueinstellungen:
1. Notieren Sie Benutzername und Passwort vom Manager
2. Geben Sie diese Anleitung weiter
3. Erklären Sie den Antragsprozess

### 4. Abwesenheitsplanung

Für die Büroplanung:
1. Öffnen Sie den **Kalender**
2. Prüfen Sie kommende Urlaube
3. Planen Sie Vertretungen

## Häufige Fragen (FAQ)

### Kann ich Anträge genehmigen?

Nein, die Office-Rolle hat nur Leserechte. Wenden Sie sich an den zuständigen Teamleiter oder Manager.

### Warum sehe ich alle Anträge?

Die Office-Rolle ist für administrative Unterstützung gedacht. Sie benötigen Einblick, um Berichte zu erstellen und Abwesenheiten zu planen.

### Kann ich Mitarbeiter anlegen?

Nein, das kann nur der Manager. Sie können aber die Daten vorbereiten und weiterleiten.

### Wer genehmigt meinen Urlaub?

Da Sie keinen Teamleiter haben, geht Ihr Antrag direkt zum Manager.
